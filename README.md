# 🎓 Student Grade Calculator

## 📌 Project Overview
This project is a simple Python program that calculates a student's grade based on marks and displays an encouraging message.

It uses basic programming concepts like:
- If-elif-else statements
- Functions
- Loops
- Input validation

---

## ⚙️ Features
- Takes student name and marks as input
- Validates input (only accepts 0–100)
- Assigns grade (A, B, C, D, F)
- Displays motivational message
- Handles invalid inputs using loops

---

## 🧠 Grading Logic
- A: 90–100
- B: 80–89
- C: 70–79
- D: 60–69
- F: 0–59
