# Student Grade Calculator

## Project Overview

This project is a simple Python program that calculates a student’s grade based on marks entered by the user. It takes input for student name and marks, then calculates the grade and displays a message accordingly.

The main aim of this project is to practice decision-making (if-elif-else) and basic programming concepts in Python.

---

## Objectives

* To calculate grades automatically based on marks
* To practice conditional statements
* To handle user input properly
* To display meaningful output

---

## Setup Instructions

1. Make sure Python is installed on your system
2. Download or clone this repository
3. Open the project folder in VS Code or terminal
4. Run the program using:
   python grade_calculator.py

## Grading Logic

The grade is assigned based on marks using if-elif-else conditions:

* 90 and above → Grade A
* 75 to 89 → Grade B
* 60 to 74 → Grade C
* 50 to 59 → Grade D
* Below 50 → Fail

---

## Input Validation

The program ensures that marks entered are between 0 and 100.
If the user enters invalid marks, it shows an error message and asks for correct input.
